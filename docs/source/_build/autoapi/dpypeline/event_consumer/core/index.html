class dpypeline.event_consumer.core.EventConsumer(queue: Queue, job_producer: JobProducer, worker: threading.Thread = None)

Event consumer that runs on a thread as a daemon process.

This event consumer consumes events from an in-memory queue and processes them to produce jobs.

Variables:

  • _queue – Queue where events are placed.

  • _job_producer – Producer of jobs.

  • optional (_worker,) – Worker thread that consumes events from the and processes them to produce jobs.

_consume_event(event) → None

Consume event by producing jobs.

Parameters:

event – Event representing file or directory creation, deletion, modification or moving.

_run_worker(sleep_time: int = 1) → None

Run the worker thread.

Callback function to be used as a target by Thread.

Parameters:

sleep_time – Sleep time in seconds for which the thread is idle.

_create_worker(daemon: bool = True) → threading.Thread

Create the worker thread.

Parameters:

daemon – If True, runs the thread as a daemon; otherwise thread is not created as a daemon.

Returns:

Worker thread that consumes events from the in-memory queue and processes them to produce jobs.

run() → None

Run the event consumer.
